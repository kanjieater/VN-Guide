# VN-Guide

A mobile-optimized, single-page application (SPA) for step-by-step Visual Novel walkthroughs. It features large touch targets and local progress tracking per route.

**Live Site:** [https://kanjieater.github.io/VN-Guide/](https://kanjieater.github.io/VN-Guide/)

## Supported Games

Currently, this app serves exclusively as a guide for the PC-98/Elf Classics version of **YU-NO: A Girl Who Chants Love at the Bound of this World** (*Kono Yo no Hate de Koi o Utau Shoujo YU-NO*). The structure is designed to support additional visual novels in the future.

The step-by-step instructions, routing logic, and UI interactions for the YU-NO guide were synthesized from the following three sources:
